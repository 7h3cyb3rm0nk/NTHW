# Not The Hidden Wiki

## Learn Rust
-----

1. [Criterion: Statistics-driven micro-benchmarking library]("https://docs.rs/criterion/latest/criterion/")
2. [cargo-fuzz: Cargo subcommand for fuzzing with libFuzzer]("https://github.com/rust-fuzz/cargo-fuzz")
3. [afl.rs: American fuzzy lop (AFL) is a popular, effective, and modern fuzz testing tool]("https://github.com/rust-fuzz/afl.rs")
4. [quickcheck: Property based testing with shrinking]("https://docs.rs/quickcheck/1.0.3/quickcheck/")
5. [proptest: Hypothesis-like property testing for Rust]("https://github.com/proptest-rs/proptest")
6. [cargo-mutants: Mutation testing]("https://docs.rs/quickcheck/1.0.3/quickcheck/")
7. [mutagen: Mutation testing]("mutagen")
8. [cargo-tarpaulin: Tool to determine code coverage achieved via tests]("https://github.com/xd009642/tarpaulin")
9. [cargo-msrv: This crate can assist you in finding the Minimum Supported Rust Version for a crate.]("https://github.com/foresterre/cargo-msrv")
10. [miri: Miri is an Undefined Behavior detection tool]("https://github.com/rust-lang/miri")
11. [cargo-hack: Cargo subcommand to provide various options useful for testing and continuous integration]("https://github.com/taiki-e/cargo-hack")
12. [loom: Loom is a tool for testing concurrent programs]("https://docs.rs/loom/latest/loom/")
13. [cargo-llvm-cov: Cargo subcommand to easily use LLVM source-based code coverage]("https://github.com/taiki-e/cargo-llvm-cov")
14. [rustfmt: A tool for formatting Rust code according to style guidelines.]("https://github.com/rust-lang/rustfmt")
15. [clippy: A collection of lints to catch common mistakes]("https://github.com/rust-lang/rust-clippy")
16. [cargo doc: A collection of lints to catch common mistakes]("https://doc.rust-lang.org/cargo/commands/cargo-doc.html")
